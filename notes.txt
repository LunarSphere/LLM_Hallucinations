Context: Setting up a conda environment (egoblind_exp1) for a research project on Palmetto 2 (Clemson's HPC cluster), working from JupyterHub on nodeai03.
 
What was tried and what worked:
 
The original environment.yaml was slow because it mixed pytorch/nvidia/conda-forge channels causing a huge solver conflict. The base conda environment on Palmetto is also broken/locked — conda install hangs indefinitely, so conda-libmamba-solver could not be installed.
 
Working solution: Created a minimal conda env with just python + pip, then installed everything else via pip:
 
    conda create -n egoblind_exp1 python=3.10 pip -y
    conda activate egoblind_exp1
    pip install --extra-index-url https://download.pytorch.org/whl/cu118 \
        torch==2.3.0+cu118 \
        torchvision==0.18.0+cu118 \
        transformers==4.46.0 \
        accelerate decord Pillow numpy pandas openai \
        tenacity einops timm sentencepiece tabulate
